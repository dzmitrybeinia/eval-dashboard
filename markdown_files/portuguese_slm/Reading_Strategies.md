# Lesson name: Estratégias de Leitura Eficazes: Identificação de Ideias Principais, Skimming, Scanning e Anotação de Textos

## Question 1: YesNo

**Statement:** A frase tópica de um parágrafo é sempre encontrada no início.

**Answer:** False

**Feedback:**
A frase tópica pode ser encontrada no início, mas também pode ser colocada no meio ou no final do parágrafo, servindo como um resumo da ideia principal. É uma estratégia para ajudar os leitores a identificar o ponto principal do texto.


## Question 2: YesNo

**Statement:** Para melhorar suas habilidades de skimming, é essencial ler cada palavra do texto com atenção.

**Answer:** False

**Feedback:**
A afirmação está incorreta porque skimming é uma técnica de leitura rápida que se concentra nas informações importantes, como títulos, subtítulos e frases iniciais e finais, sem ler cada palavra em detalhe.


## Question 3: YesNo

**Statement:** O scanning é uma maneira eficiente de encontrar informações importantes rapidamente.

**Answer:** True

**Feedback:**
Esta afirmação é verdadeira porque o scanning é uma habilidade de leitura que permite ao leitor localizar informações específicas em um texto sem ler cada palavra. É útil quando se precisa de dados ou fatos rápido, como uma data ou um nome.


## Question 4: YesNo

**Statement:** A anotação de textos é melhor feita sem destaque ou sublinhado.

**Answer:** False

**Feedback:**
A anotação de textos envolve a identificação de ideias principais e detalhes importantes para melhorar a compreensão. Destaques e sublinhados são ferramentas comuns para destacar esses elementos críticos, não para evitar a ativação deles.


## Question 5: YesNo

**Statement:** No processo de leitura, identificar ideias principais é menos importante do que ler cada palavra do texto.

**Answer:** False

**Feedback:**
Identificar ideias principais é crucial para entender o núcleo do texto. Ler cada palavra é uma estratégia que pode ser usada em leituras mais detalhadas, mas quando se busca compreender o significado central de um texto, focar nas ideias principais é mais eficiente.
